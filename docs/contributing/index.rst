Local setup quick start
-----------------------

1. Create or activate your Python environment.
2. Install pyosMeta in editable mode with development dependencies:

   .. code-block:: console

      pip install -e ".[dev]"

3. Run tests:

   .. code-block:: console

      hatch run test:run-coverage
