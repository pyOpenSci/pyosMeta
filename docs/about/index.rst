About
=====

pyosMeta is the package used by pyOpenSci to update contributor metadata and
peer-review package metadata that powers sections of the pyOpenSci website.

Current CLI entry points include:

* ``parse-history``
* ``update-contributors``
* ``update-reviews``
* ``update-review-teams``
* ``fetch-rss-feed``
